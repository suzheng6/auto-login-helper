# 自动登录小帮手

Telegram 自动登录工具，支持批量账号处理和 AyuGram tdata 写入。

## 功能

- Telethon API 自动登录（验证码 + 2FA 从 URL 自动获取）
- 批量账号连续处理
- AyuGram tdata 多账号写入
- 使用 TelegramDesktop 官方 API 凭证，避免账号冻结
- 每个账号独立设备指纹
- 启动时自动检查更新
- 中英文双语界面
- F3 开始 / F5 紧急停止

## 使用方法

1. 从 [Releases](https://github.com/suzheng6/auto-login-helper/releases) 下载最新版 EXE
2. 双击运行，无需安装 Python 或任何依赖
3. 在文本框中粘贴账号信息（格式：`手机号|验证码URL`）
4. 选择要登录的Ayugram所在的位置
5. 按 F3 开始自动登录

## 更新

程序启动时会自动检查 GitHub 上的最新版本，如有更新会弹窗提示一键更新。
